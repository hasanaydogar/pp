# Environment Variables Kontrol Listesi

`.env.local` dosyanızda şu değişkenler olmalı:

```env
NEXT_PUBLIC_SUPABASE_URL=https://pnmisbgmzdceaoysmdbdc.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
SUPABASE_SERVICE_ROLE_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

## Kontrol:

1. ✅ **NEXT_PUBLIC_SUPABASE_URL** - Supabase Project URL'iniz
2. ✅ **NEXT_PUBLIC_SUPABASE_ANON_KEY** - anon/public key
3. ✅ **SUPABASE_SERVICE_ROLE_KEY** - service_role key (gördüğüm kadarıyla var ✅)
4. ✅ **NEXT_PUBLIC_SITE_URL** - http://localhost:3000

## Önemli:

- Dosyayı **kaydettiğinizden** emin olun
- Dosya proje kök dizininde olmalı (`personal-portfoy/.env.local`)
- Dosya adı tam olarak `.env.local` olmalı (başında nokta var!)
